Concept

SATS is an automated assessment system used in the context of computer programming excercises. As an undergraduate student reserach assitant I wireframed the user interfaces, primarily for students, and programmed the JavaScript framework. Object Oriented programming concepts were used and coded in Javascript to allow for the website to load faster.
